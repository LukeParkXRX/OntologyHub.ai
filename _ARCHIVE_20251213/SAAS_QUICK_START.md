# OntologyHub.AI SaaS 서비스 - 빠른 시작 가이드

## 🎯 개요

OntologyHub.AI가 이제 SaaS 플랫폼으로 업그레이드되었습니다!

**새로운 기능:**
- ✅ 도메인별 데이터 수집 및 관리
- ✅ 자동 지식 그래프 생성
- ✅ 다양한 형식으로 데이터 내보내기 (JSON, CSV, GraphML)
- ✅ 멀티테넌트 격리 (사용자별 데이터 분리)
- ✅ 구독 플랜 기반 제한

---

## 📦 설치 및 실행

### 1. 데이터베이스 설정

Supabase SQL 에디터에서 다음 파일들을 순서대로 실행하세요:

1. `supabase_schema.sql` (기존 스키마, 이미 실행했다면 건너뛰기)
2. `supabase_saas_schema.sql` (새로운 SaaS 스키마) ⭐

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속

## 🚀 사용 방법

### 1단계: 도메인 생성

1. 상단 메뉴에서 "My Domains" 클릭
2. "New Domain" 버튼 클릭
3. 정보 입력:
   - **이름**: 예) "의료 연구"
   - **카테고리**: Healthcare, Finance, Education 등
   - **설명**: 도메인 설명 (선택사항)
4. "Create Domain" 클릭

### 2단계: 데이터 수집

1. 생성한 도메인 카드 클릭
2. "Add Data" 버튼 클릭
3. 데이터 입력:
   - **제목**: 데이터 항목 제목
   - **내용**: 지식 그래프로 변환할 텍스트
   - **자동 그래프 생성**: 체크 (권장)
4. "Add Entry" 클릭
5. AI가 자동으로 지식 그래프 생성 (2-3초 소요)

### 3단계: 데이터 내보내기

1. 도메인 상세 페이지에서 "Export Data" 클릭
2. 내보내기 형식 선택:
   - **JSON**: 완전한 데이터 (메타데이터 포함)
   - **CSV**: 스프레드시트용 (Excel, Google Sheets)
   - **GraphML**: 그래프 분석 도구용 (Gephi, yEd, Cytoscape)
3. "Include Knowledge Graphs" 옵션 선택
4. "Export" 클릭하여 다운로드

## 🎨 주요 기능

### 도메인 관리
- 여러 도메인 생성 및 관리
- 카테고리별 색상 구분
- 도메인별 통계 (데이터 개수, 그래프 개수)
- 도메인 삭제 (모든 데이터 함께 삭제)

### 데이터 수집
- 텍스트 입력으로 데이터 추가
- AI 자동 지식 그래프 생성
- 데이터 항목별 그래프 미리보기
- 개별 데이터 삭제

### 데이터 내보내기
- 3가지 형식 지원 (JSON, CSV, GraphML)
- 그래프 포함/제외 선택 가능
- 즉시 다운로드
- 내보내기 이력 추적

---

## 🔒 보안 및 제한

### 멀티테넌트 격리
- 사용자는 자신의 데이터만 접근 가능
- 데이터베이스 레벨에서 격리 (RLS)
- API를 통한 우회 불가능

### 무료 플랜 제한
- 최대 3개 도메인
- 도메인당 최대 50개 데이터 항목
- JSON 형식만 내보내기 가능
- 내보내기 파일 크기 10MB 제한

## 🐛 문제 해결

### 도메인이 생성되지 않음
- 로그인 상태 확인
- 도메인 개수 제한 확인 (무료: 3개)
- 브라우저 콘솔에서 에러 확인

### 그래프가 생성되지 않음
- OpenAI API 키 확인
- 텍스트 길이 확인 (너무 짧으면 그래프 생성 어려움)
- 네트워크 연결 확인

### 내보내기가 실패함
- 파일 크기 제한 확인 (무료: 10MB)
- 내보내기 형식 권한 확인 (무료: JSON만)
- 브라우저 다운로드 설정 확인
