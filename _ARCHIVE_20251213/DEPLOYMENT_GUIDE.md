# 배포 가이드 (Deployment Guide)

OntologyHub.AI의 변경 사항을 실제 웹사이트(라이브 서버)에 적용하는 방법입니다.

## 🚀 수동 배포 (현재 방식)

터미널에서 다음 명령어를 입력하면 현재 컴퓨터의 코드가 Vercel 서버로 전송되어 배포됩니다.

```bash
npx vercel --prod
```

*   **`npx vercel`**: 테스트(Preview) 서버에 배포합니다. (주로 개발자 확인용)
*   **`npx vercel --prod`**: 실제 사용자에게 보여지는 **운영(Production)** 서버에 배포합니다.

## 🔄 자동 배포 (권장 방식)

프로젝트를 **GitHub**에 업로드하고 Vercel과 연동하면 훨씬 편하게 관리할 수 있습니다.

1.  코드를 수정하고 GitHub에 `push` 합니다.
2.  Vercel이 자동으로 변경 사항을 감지하여 배포를 시작합니다.
3.  별도의 명령어 입력 없이 약 1~2분 뒤에 사이트가 업데이트됩니다.

---

### 💡 팁
*   배포 후에는 브라우저 캐시 때문에 디자인이 바로 바뀌지 않아 보일 수 있습니다. 그럴 땐 `Ctrl + Shift + R` (Mac은 `Cmd + Shift + R`)을 눌러 **강력 새로고침**을 해보세요.
